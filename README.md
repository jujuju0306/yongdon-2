# 용동 🐷 - 용돈 관리 PWA

고등학생을 위한 모바일 최적화 용돈 관리 Progressive Web App

## 주요 기능
- 📊 대시보드: 도넛 차트 + 프로그레스 바 + 지난달 vs 이번달 비교
- 📝 기입장: 카테고리별 소비 내역 기록 및 관리
- ⭐ 마일리지: 절약할수록 캐릭터가 성장하는 리워드 시스템
- 🚨 경고 알람: 예산 초과 시 실시간 경고 배너
- 🛒 위시리스트: 3일 쿨다운 충동구매 방지 기능

## 기술 스택
- HTML5 + Tailwind CSS v3 CDN
- Vanilla JavaScript
- Chart.js
- PWA (manifest.json + Service Worker)
- localStorage (데이터 영속화)

## 실행 방법
`index.html` 파일을 브라우저에서 열면 바로 사용 가능합니다.
